# Product Roadmap

This is nexu's public-facing roadmap.

We update it with every release. If you have feature suggestions or want to weigh in on priorities, join the conversation at [GitHub Discussions](https://github.com/nexu-io/nexu/discussions).

---

## ✅ Shipped

**Windows official release**
Windows 10+ users can now use nexu with full feature parity to the macOS version.

**WeChat**
Connect via the WeChat 8.0.7 OpenClaw plugin — scan, connect, and chat with your AI Agent from your phone, no extra setup required.

**Telegram and WhatsApp**
Connect your Agent to a Telegram Bot or WhatsApp — great for international users and global teams.

**WeCom, DingTalk, and QQ**
Full coverage of China's major enterprise IM platforms. Whatever tool your team is on, you can plug right in.

**One-click login for MiniMax, Codex, and GLM**
MiniMax, OpenAI Codex, and GLM (Z.AI) support OAuth login — authorize with a single click, no API key copy-paste needed.

**Local Ollama models**
Connect to a locally deployed Ollama model. Your data never leaves your machine.

**Credits and rewards system**
Earn credits through daily check-ins, sharing, and invites. Complete tasks like GitHub Star or posting on X for bonus credits.

**System proxy auto-detection**
nexu reads your system proxy settings automatically — nothing to configure manually, just works.

---

## 🚧 Building

**Conversation history survives upgrades** (Windows)
After a Windows update, your previous conversations are automatically restored — no manual recovery needed.

**Installed Skills survive upgrades** (Windows)
Skills you've installed stay installed after an upgrade. No need to set them up again.

**Readable error messages for channel failures**
When a channel fails to connect, you'll see a clear explanation of why — not a blank screen or a cryptic error code.

**More intuitive conversation names**
Direct chats show the contact's name; group chats show the group name. Finding the right conversation is instant.

**Global keyboard shortcut**
Bring up nexu from anywhere with a keyboard shortcut — no need to switch apps manually.

**Credits stay in sync across web and desktop**
Your credit balance shows the same number everywhere, updated in real time.

---

## 📋 Planned

**Complete OAuth authorization inside IM**
Authorize new integrations directly in the chat window — no browser redirect.

**In-app update notes**
See what's new in each release right inside the client, without hunting through release notes.

**Long conversations stay snappy**
Response speed stays smooth as conversations grow — no more slowdowns in long threads.

**Password reset via email**
Reset your password from the login page without contacting support.

**Read status synced across web and desktop**
Messages you've read on one client stay read everywhere. No more duplicate unread notifications when switching devices.

**Custom download path for attachments**
Set a default save location for files — stop being prompted every time.

**Offline indicator**
When your network drops, nexu shows a clear offline status so you're never left guessing.

**Browse and install community Skills**
Discover Skills from other Agents directly in the client. Install in one click and start using immediately.

**Cross-agent collaboration (ACP protocol)**
Connect your Agent to others via the Agent Communication Protocol for multi-agent workflows.

**Linux support**
We're evaluating feasibility. Tell us about your use case in [GitHub Issues](https://github.com/nexu-io/nexu/issues) — demand shapes priority.
