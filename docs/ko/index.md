# 소개

**nexu** (next to you)는 **OpenClaw 🦞** Agent를 Feishu, Slack, Discord 및 기타 IM 채널에 연결하는 오픈소스 데스크톱 클라이언트(Electron)입니다.

그래픽 설정, 내장 Feishu Skills, 다중 모델 지원, BYOK를 제공합니다.

설정과 런타임 상태는 주로 사용자의 기기에 저장됩니다.
