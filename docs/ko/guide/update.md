# 업데이트 가이드

## 방법 1: 클라이언트 내에서 업데이트

1. nexu를 엽니다
2. 상단 메뉴 바에서 **Nexu**를 클릭합니다
3. **Check for Updates**를 선택합니다
4. 새 버전이 있으면 안내를 따릅니다

> 가장 쉬운 방법이며, 권장됩니다.

![업데이트 확인](/assets/nexu-check-for-updates.webp)
