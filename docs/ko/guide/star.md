## Star를 눌러야 하는 이유

- **가시성** — Star는 더 많은 개발자와 팀이 nexu를 찾는 데 도움이 됩니다
- **동기 부여** — 모든 Star는 팀이 계속 개선하도록 격려합니다
- **나중에 쉽게 찾기** — Star한 저장소는 GitHub Stars 목록에 남아 있습니다
