# 1분 빠른 시작

## 1단계: 다운로드 및 설치

[nexu 클라이언트](https://nexu.io/download)를 다운로드하고, 설치한 후 더블 클릭하여 실행하세요.

![nexu 홈 화면](/assets/nexu-home.webp)

## 2단계: 모드 선택

왼쪽 사이드바에서 **설정**을 열고 통합 모드를 선택하세요:

- **로그인** — **Nexu Official**을 선택하고 **Sign in to Nexu**를 클릭하면 고품질 모델을 즉시 사용할 수 있습니다
- **BYOK 모드** — Anthropic 또는 OpenAI 등 프로바이더를 선택하고, API 키를 붙여넣은 후 **Save**를 클릭하세요

![모델 통합 모드 선택](/assets/nexu-settings-open.webp)

![Nexu Official 로그인](/assets/nexu-models-official.webp)

## 3단계: 채널 설정

채널 하나를 선택하고 설정을 완료하세요:

- [Feishu](/ko/guide/channels/feishu) — App ID + App Secret만 있으면 됩니다
- [Slack](/ko/guide/channels/slack) — 매니페스트를 통한 원클릭 앱 생성
- [Discord](/ko/guide/channels/discord) — Application ID + Bot Token 필요

설정이 완료되면 **Chat**을 클릭하여 Agent와 대화를 시작하세요.
