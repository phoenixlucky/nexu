# 공유로 크레딧 받기

`nexu`는 추가 포인트 보상 시스템을 제공합니다. 일일 포인트 드롭, 오픈소스 프로젝트 지원, 소셜 플랫폼에서의 `nexu` 공유를 통해 추가 포인트를 얻을 수 있습니다.

## 받을 수 있는 포인트

현재 사용 가능한 보상:

- 일일 포인트: +100 포인트, 매일 `nexu` 계정으로 자동 지급되며 다음 날 자정에 만료
- GitHub Star: +700 포인트, 1회 한정
- X에 공유: +200 포인트, 주 1회 가능
- Reddit에 게시: +200 포인트, 주 1회 가능
- LinkedIn에 게시: +200 포인트, 주 1회 가능
- Facebook에 공유: +200 포인트, 주 1회 가능
- WhatsApp에 공유: +200 포인트, 주 1회 가능
- 모바일에서 `nexu` GitHub 저장소를 Feishu, WeChat 등 앱에 공유: +200 포인트, 주 1회 가능

## 포인트 받는 방법

`nexu` 클라이언트의 `Rewards` 페이지에서 모든 보상 작업과 현재 포인트 상태를 확인할 수 있습니다.

작업별 방법은 다음과 같습니다.

### 1. 일일 포인트는 자동으로 지급됩니다

일일 포인트는 매일 `nexu` 계정으로 자동 발송됩니다. 수동 체크인이나 수령이 필요 없습니다.

### 2. GitHub에서 Star 하기

작업의 GitHub 링크를 열고 `nexu` 저장소에 Star를 한 후, `Rewards` 페이지로 돌아와 확인을 완료하세요.

저장소: [nexu-io/nexu](https://github.com/nexu-io/nexu)

### 3. 소셜 플랫폼에서 nexu 공유

X, Reddit, LinkedIn, Facebook 또는 WhatsApp에서 `nexu`를 공유할 수 있습니다. 공유 후, `Rewards` 페이지로 돌아가 안내에 따라 보상을 수령하세요.

모바일에서 `nexu` GitHub 저장소를 Feishu나 WeChat 등 앱에 직접 공유할 수도 있습니다. 이것도 보상 포인트 대상입니다.

## 참고사항

- 일일 포인트는 자동 지급되므로 수동 체크인 불필요
- `GitHub Star` 보상은 1회만 수령 가능
- 소셜 공유 작업은 현재 주간 작업
- Feishu, WeChat 등 앱으로의 모바일 GitHub 공유 작업도 주간 작업

## 작업 갱신 시간

- 일일 작업: 매일 UTC 0:00에 갱신
- 주간 작업: 소셜 공유 작업은 매주 월요일 UTC 0:00에 갱신

## 포인트 만료

- 일일 포인트: 당일 UTC 23:59:59에 만료
- 소셜 공유 포인트: 7일 후 UTC 23:59:59에 만료
- GitHub Star 포인트: 1개월 후 UTC 23:59:59에 만료
- 가입 보너스 포인트: 1개월 후 UTC 23:59:59에 만료

## FAQ

**Q: 작업을 완료했는데 포인트가 바로 지급되지 않는 이유는?**

소셜 공유와 `GitHub Star` 작업은 보통 `Rewards` 페이지에서 확인이 필요합니다. 포인트가 바로 표시되지 않으면 페이지를 새로고침하거나 나중에 다시 시도하세요.

**Q: 일일 포인트를 받으려면 수동으로 체크인해야 하나요?**

아니요. 일일 포인트는 매일 `nexu` 계정으로 자동 발송됩니다.

**Q: 주간 작업은 언제 갱신되나요?**

소셜 공유 작업은 매주 월요일 UTC 0:00에 갱신됩니다.

**Q: 가입 보너스는 신규 사용자만 받을 수 있나요?**

아니요. 기존 가입 사용자도 300 보너스 포인트를 받습니다.

**Q: 소셜 공유 보상을 다시 받을 수 있나요?**

대부분의 소셜 공유 보상은 주 1회 수령 가능합니다. 다음 주기가 시작되면 작업을 다시 완료하고 보상을 다시 수령할 수 있습니다.
