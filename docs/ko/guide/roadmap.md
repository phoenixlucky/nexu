# 제품 로드맵

nexu의 공개 로드맵입니다.

매 릴리스마다 업데이트됩니다. 기능 제안이나 우선순위에 대한 의견이 있으시면 [GitHub Discussions](https://github.com/nexu-io/nexu/discussions)에서 알려주세요.

---

## ✅ 출시 완료

**Windows 정식 버전**
Windows 10+ 사용자도 macOS 버전과 동일한 기능으로 nexu를 사용할 수 있습니다.

**WeChat 연동**
WeChat 8.0.7 OpenClaw 플러그인으로 스캔 후 연결하면 끝. 추가 설정 없이 스마트폰에서 AI Agent와 대화할 수 있습니다.

**Telegram과 WhatsApp**
Agent를 Telegram Bot 또는 WhatsApp에 연결할 수 있습니다. 해외 사용자와 글로벌 팀에 적합합니다.

**WeCom·DingTalk·QQ 지원**
중국 주요 기업용 IM 플랫폼 전체를 지원합니다. 팀에서 어떤 도구를 사용하든 바로 연결할 수 있습니다.

**MiniMax·Codex·GLM 원클릭 로그인**
MiniMax, OpenAI Codex, GLM(Z.AI)이 OAuth 로그인을 지원합니다. 클릭 한 번으로 인증하며, API Key를 복사할 필요가 없습니다.

**Ollama 로컬 모델 지원**
로컬에 배포된 Ollama 모델에 연결할 수 있습니다. 데이터는 내 기기 밖으로 나가지 않습니다.

**크레딧 및 리워드 시스템**
매일 체크인, 공유, 초대로 크레딧을 획득하세요. GitHub Star나 X 게시 등의 태스크 완료 시에도 보상을 받을 수 있습니다.

**시스템 프록시 자동 감지**
시스템 프록시 설정을 자동으로 읽어옵니다. 수동 설정 없이 프록시 환경에서도 바로 사용할 수 있습니다.

## 🚧 개발 중

**업그레이드 후 대화 기록 자동 복원** (Windows)
Windows에서 버전 업그레이드 후에도 이전 대화가 자동으로 복원됩니다. 수동 복구가 필요 없습니다.

**업그레이드 후 설치된 Skills 자동 복원** (Windows)
설치된 Skills가 업그레이드 후에도 유지됩니다. 매번 다시 설치할 필요가 없습니다.

**채널 연결 실패 시 명확한 오류 메시지**
채널 연결에 실패하면 빈 화면이나 알 수 없는 오류 코드 대신 명확한 원인을 표시합니다.

**더 직관적인 대화 이름**
개인 채팅에는 상대방 이름, 그룹 채팅에는 그룹 이름이 표시됩니다. 원하는 대화를 빠르게 찾을 수 있습니다.

**전역 키보드 단축키**
키보드 단축키로 nexu를 즉시 불러올 수 있습니다. 앱을 수동으로 전환할 필요가 없습니다.

**웹과 데스크톱 크레딧 실시간 동기화**
어디서 확인해도 동일한 잔액이 실시간으로 표시됩니다.

## 📋 계획 중

**IM 내에서 OAuth 인증 완료**
브라우저로 이동하지 않고 채팅 창 안에서 OAuth 인증을 완료할 수 있습니다.

**앱 내 업데이트 내용 표시**
업그레이드할 때마다 클라이언트 내에서 변경 사항을 바로 확인할 수 있습니다. Release Notes를 직접 찾아볼 필요가 없습니다.

**긴 대화에서도 빠른 응답 유지**
대화가 길어져도 응답 속도가 느려지지 않습니다.

**이메일로 비밀번호 재설정**
로그인 페이지에서 이메일로 비밀번호를 재설정할 수 있습니다. 지원팀에 문의할 필요가 없습니다.

**읽음 상태 크로스 디바이스 동기화**
한 클라이언트에서 읽은 메시지는 다른 클라이언트에서도 읽음으로 처리됩니다. 기기를 전환해도 중복 알림이 표시되지 않습니다.

**첨부 파일 다운로드 경로 커스텀**
파일의 기본 저장 위치를 설정할 수 있습니다. 매번 폴더를 선택할 필요가 없습니다.

**오프라인 상태 전역 표시**
네트워크가 끊기면 인터페이스에 명확한 오프라인 상태가 표시됩니다.

**커뮤니티 Skills 탐색 및 설치**
다른 Agent의 Skills를 클라이언트 내에서 직접 발견하고 원클릭으로 설치해 바로 사용할 수 있습니다.

**Agent 간 협업 (ACP 프로토콜)**
Agent Communication Protocol을 통해 내 Agent가 다른 Agent와 협력할 수 있게 됩니다.

**Linux 지원**
실현 가능성을 검토 중입니다. [GitHub Issues](https://github.com/nexu-io/nexu/issues)에서 사용 사례를 알려주세요. 수요가 우선순위를 결정합니다.
