# 如何通过分享 nexu 获得积分

`nexu` 提供了一套额外积分奖励机制。你可以通过每日积分自动发放、支持开源项目，以及在社交平台分享 `nexu`，获得额外积分，用于继续体验更多能力。

## 你可以获得哪些积分

目前可领取的积分如下：

- 每日积分：+100 积分，次日凌晨失效，每天凌晨自动发放到 `nexu` 账户
- GitHub Star：+700 积分，仅限一次
- 分享到 X：+200 积分，每周可领取一次
- 发帖到 Reddit：+200 积分，每周可领取一次
- 发帖到领英（LinkedIn）：+200 积分，每周可领取一次
- 分享到 Facebook：+200 积分，每周可领取一次
- 分享到 WhatsApp：+200 积分，每周可领取一次
- 在移动端分享 `nexu` 的 GitHub 项目到飞书、微信等平台：+200 积分，每周可领取一次

## 如何领取积分

你可以在 `nexu` 客户端的 `Rewards` 页面查看全部奖励任务和积分状态。

不同任务的领取方式如下。

### 1. 每日积分自动发放

每日积分会在每天凌晨自动发放到你的 `nexu` 账户，不需要额外打卡或手动领取。

### 2. 在 GitHub 上为我们点 Star

打开任务中的 GitHub 链接，给 `nexu` 仓库点亮 Star，然后回到 `Rewards` 页面完成确认。

仓库地址：[nexu-io/nexu](https://github.com/nexu-io/nexu)

### 3. 在社交平台分享 nexu

你可以在 X、Reddit、领英、Facebook 或 WhatsApp 等平台分享 `nexu`。完成分享后，回到 `Rewards` 页面按提示确认，即可领取对应积分。

另外，你也可以在移动端直接分享 `nexu` 的 GitHub 项目到飞书、微信等平台，完成后同样可以获得对应积分。

## 注意事项

- 每日积分自动发放，不需要手动打卡领取
- `GitHub Star` 奖励只能领取一次
- 社交分享任务目前按周领取
- 移动端分享 GitHub 项目到飞书、微信等平台的任务，同样按周领取

## 积分任务刷新时间

- 天级别任务：每天 UTC 0 点刷新
- 周级别任务：社交分享任务每周一 UTC 0 点刷新

## 积分过期时间

- 每日积分：当天 UTC 23:59:59 过期
- 社交分享积分：7 天后的 UTC 23:59:59 过期
- GitHub Star 积分：一个月后的 UTC 23:59:59 过期
- 注册赠送积分：一个月后的 UTC 23:59:59 过期

## 常见问题

**Q: 为什么我完成了任务，但没有马上拿到积分？**

社交分享和 `GitHub Star` 任务通常还需要回到 `Rewards` 页面按提示确认。若仍未到账，可以先刷新页面或稍后再试。

**Q: 每日积分需要我手动打卡吗？**

不需要。每日积分会在每天凌晨自动发放到你的 `nexu` 账户。

**Q: 每周任务什么时候刷新？**

社交分享任务会在每周一 UTC 0 点刷新。

**Q: 注册赠送积分是只有新用户才有吗？**

不是，已注册用户也会赠送的 300 积分。

**Q: 社交平台分享任务可以重复领取吗？**

大多数社交分享任务是“每周一次”。进入新的可领取周期后，再按要求完成一次即可再次领取。
