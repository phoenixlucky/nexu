# 介绍

**nexu**（next to you）让你用最简单的方式拥有一个 OpenClaw 🦞——下载、双击、开始工作。

基于开源 OpenClaw 🦞 打造，预装飞书 Skills 等关键能力，内置 Claude、ChatGPT、Minimax、GLM、Kimi 等主流模型，也支持自带 API Key 🔌。

连接飞书、Slack 或 Discord，Agent 直达手机端 📱——在你日常用的聊天工具里直接对话。
