# プロダクトロードマップ

これは nexu の公開ロードマップです。

リリースごとに更新されます。機能の提案や優先度についてのご意見は、[GitHub Discussions](https://github.com/nexu-io/nexu/discussions) でお知らせください。

---

## ✅ リリース済み

**Windows 正式版**
Windows 10+ ユーザーも、macOS 版と同等の機能で nexu を利用できるようになりました。

**WeChat 対応**
WeChat 8.0.7 の OpenClaw プラグインを使ってスキャン・接続するだけ。追加設定不要で、スマートフォンから AI Agent とチャットできます。

**Telegram と WhatsApp**
Agent を Telegram Bot や WhatsApp に接続できます。海外ユーザーやグローバルチームでも快適に利用できます。

**WeCom・DingTalk・QQ 対応**
中国主要エンタープライズ IM プラットフォームに対応。チームのツールを問わず接続できます。

**MiniMax・Codex・GLM のワンクリックログイン**
MiniMax、OpenAI Codex、GLM（Z.AI）が OAuth ログインに対応。ワンクリックで認可でき、API Key の手入力は不要です。

**Ollama ローカルモデル対応**
ローカルにデプロイした Ollama モデルに接続できます。データは端末の外に出ません。

**クレジット・リワードシステム**
毎日のチェックイン、シェア、招待でクレジットを獲得。GitHub Star や X への投稿などのタスク完了でも報酬が得られます。

**システムプロキシの自動検出**
システムのプロキシ設定を自動的に読み込みます。手動設定は不要で、プロキシ環境でもそのまま動作します。

## 🚧 開発中

**アップグレード後に会話履歴を自動復元**（Windows）
Windows でバージョンアップ後も、以前の会話を自動的に復元。手動での回復は不要になります。

**アップグレード後にインストール済み Skills を自動復元**（Windows）
インストール済みの Skills はアップグレード後も保持。毎回再設定する手間がなくなります。

**チャンネル接続失敗時にわかりやすいエラー表示**
チャンネルの接続に失敗した際、空白画面や不可解なエラーコードではなく、原因を明確に表示します。

**会話名をより直感的に**
個人チャットには相手の名前、グループチャットにはグループ名を表示。目的の会話をすぐに見つけられます。

**グローバルキーボードショートカット**
ショートカットキーで nexu をすぐに呼び出せます。アプリを手動で切り替える必要がなくなります。

**Web とデスクトップのクレジットをリアルタイム同期**
どこで確認しても同じ残高が表示されるようになります。

## 📋 計画中

**IM 内で OAuth 認可を完結**
ブラウザにリダイレクトすることなく、チャットウィンドウ内で OAuth 認可を完了できます。

**アプリ内でアップデート内容を確認**
アップグレードのたびに、クライアント内で変更点をすぐに確認できます。Release Notes を手動で調べる必要がなくなります。

**長い会話でもスムーズな動作を維持**
会話が長くなっても応答速度が落ちなくなります。

**メールによるパスワードリセット**
ログインページからメールでパスワードをリセットできます。サポートへの連絡は不要です。

**既読状態のクロスデバイス同期**
一方のクライアントで既読にしたメッセージは、もう一方でも既読として扱われます。デバイスを切り替えても未読通知が重複しません。

**添付ファイルのダウンロード先をカスタマイズ**
ファイルのデフォルト保存先を設定できます。毎回フォルダを選ぶ手間がなくなります。

**オフライン時のグローバル通知**
ネットワークが切断された際、インターフェース上で明確にオフライン状態が表示されます。

**コミュニティ Skills の閲覧・インストール**
他の Agent の Skills をクライアント内から直接発見し、ワンクリックでインストールして利用できます。

**Agent 間連携（ACP プロトコル）**
Agent Communication Protocol を介して、自分の Agent と他の Agent が連携できるようになります。

**Linux サポート**
実現可能性を検討中です。ユースケースは [GitHub Issues](https://github.com/nexu-io/nexu/issues) でお知らせください。需要が優先度を決めます。
