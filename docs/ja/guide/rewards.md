# nexu をシェアしてポイントを獲得する方法

`nexu` には追加ポイントを獲得できる報酬制度があります。毎日の自動付与ポイント、オープンソースプロジェクトの応援、そして SNS での `nexu` のシェアによって、追加ポイントを受け取れます。

## 獲得できるポイント

現在受け取れるポイントは次のとおりです。

- 毎日ポイント: +100 ポイント。毎日 `nexu` アカウントに自動付与され、翌日深夜に失効
- GitHub Star: +700 ポイント、1 回限り
- X でシェア: +200 ポイント、週 1 回まで
- Reddit に投稿: +200 ポイント、週 1 回まで
- LinkedIn に投稿: +200 ポイント、週 1 回まで
- Facebook でシェア: +200 ポイント、週 1 回まで
- WhatsApp でシェア: +200 ポイント、週 1 回まで
- モバイル端末から `nexu` の GitHub リポジトリを Feishu、WeChat などへ共有: +200 ポイント、週 1 回まで

## ポイントの受け取り方法

`nexu` クライアントの `Rewards` ページで、すべての報酬タスクと現在のポイント状況を確認できます。

各タスクの受け取り方法は次のとおりです。

### 1. 毎日ポイントは自動付与

毎日ポイントは毎日自動的に `nexu` アカウントへ付与されます。手動チェックインや受け取り操作は不要です。

### 2. GitHub で Star する

タスク内の GitHub リンクを開き、`nexu` リポジトリに Star を付けてから、`Rewards` ページに戻って確認を完了してください。

リポジトリ: [nexu-io/nexu](https://github.com/nexu-io/nexu)

### 3. SNS で nexu をシェアする

`nexu` は X、Reddit、LinkedIn、Facebook、WhatsApp などでシェアできます。シェア後、`Rewards` ページに戻って案内に従えば、対応するポイントを受け取れます。

また、モバイル端末から `nexu` の GitHub リポジトリを Feishu や WeChat などのアプリへ直接共有した場合も、同様にポイント対象になります。

## 注意事項

- 毎日ポイントは自動付与のため、手動チェックインは不要です
- `GitHub Star` 報酬は 1 回しか受け取れません
- SNS シェア系タスクは現在週単位のタスクです
- Feishu、WeChat などへのモバイル GitHub 共有タスクも週単位です

## タスクの更新時間

- 日次タスク: 毎日 UTC 0:00 に更新
- 週次タスク: SNS シェアタスクは毎週月曜日 UTC 0:00 に更新

## ポイントの有効期限

- 毎日ポイント: 当日 UTC 23:59:59 に失効
- SNS シェアポイント: 7 日後の UTC 23:59:59 に失効
- GitHub Star ポイント: 1 か月後の UTC 23:59:59 に失効
- 登録ボーナスポイント: 1 か月後の UTC 23:59:59 に失効

## よくある質問

**Q: タスクを完了したのに、すぐポイントが反映されません。**

SNS シェアと `GitHub Star` タスクは、通常 `Rewards` ページに戻って確認する必要があります。すぐに反映されない場合は、ページを再読み込みするか、少し時間をおいて再度お試しください。

**Q: 毎日ポイントは手動チェックインが必要ですか？**

いいえ。毎日ポイントは毎日自動で `nexu` アカウントに付与されます。

**Q: 週次タスクはいつ更新されますか？**

SNS シェアタスクは毎週月曜日 UTC 0:00 に更新されます。

**Q: 登録ボーナスポイントは新規ユーザーだけですか？**

いいえ。すでに登録済みのユーザーにも 300 ポイントが付与されます。

**Q: SNS シェア報酬は繰り返し受け取れますか？**

多くの SNS シェア報酬は週 1 回です。次の受け取り可能な周期に入れば、再度タスクを完了して受け取れます。
